## 📌 Overview

**Neo** — bu bank call center agentlari uchun real-time AI yordamchi tizim. Agent mijoz bilan telefonda gaplashayotgan paytda, tizim suhbatni tahlil qilib, **1–2 soniya ichida** ekranda mos javoblar, tavsiyalar va mahsulot ma'lumotlarini ko'rsatadi.

---

## 💡 Muammo va Yechim

Bank agentlari ko'pincha mijoz savollariga tez va to'g'ri javob berishda qiynaladi — bu savdo samaradorligini pasaytiradi.

### Bizning yechim:

```
🎧 STT  →  🧠 LLM + RAG  →  ⚡ Overlay UI
Suhbat      Kontekst           Agent ekranida
eshitiladi  tahlil qilinadi    tavsiya paydo bo'ladi
```

| Qadam | Texnologiya | Natija |
|-------|-------------|--------|
| 🎧 Real-time audio | STT model | Suhbat matnga aylanadi |
| 🧠 Tahlil | LLM + RAG | Kontekst tushuniladi |
| ⚡ Ko'rsatish | Overlay UI | Agent 1–2 sek ichida javob oladi |

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-repo-link.git
cd sale-assistant-bot
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run backend

```bash
uvicorn main:app --reload
```

### 5. Run frontend

```bash
cd frontend
npm install
npm run dev
```
